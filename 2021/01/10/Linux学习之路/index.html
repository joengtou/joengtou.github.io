五. 权限管理

ACL权限

开启ACL

查看分区挂载：df -h
查看分区ACL权限是否开启：dumpe2fs -h /dev/vda1
临时开启分区ACL权限：mount -o remount,acl /
永久开启：修改开机默认挂载/etc/fstab 添加acl ;重新挂载文件系统mount -o remount / 或重启系统

查看与设定

setfacl -m (表示添加acl) u:用户名:权限 目录 ;g:组名
getfacl 目录名

最大有效权限与删除

mask权限类似于子网掩码,修改 setfacl -m m:权限,只影响acl用户和所属组
删除：setfacl -b 目录名 ；setfacl -x 指定组或用户名

默认与递归

递归(只能作用于现有的文件)： -R 只能放在最后
默认(作用于未来，不作用于现有文件)：setfacl -m d:u:权限 -R 目录名

文件特殊权限

setUID

必须是可执行二进制文件
命令执行者要对该程序具有x执行权限
chmod u+s chmod 4755
使用户使用文件时短暂具有所有者的权限

setGID

可执行文件或目录
chmod g+s chmod 2755

Sticky BIT

只针对目录
普通用户对该目录拥有wx权限
chmod o+t chmod 1755
只能删除自己创建的文件

文件系统属性权限chattr

对root同样有效
chattr -i 文件，文件只读
chattr -i 目录，只能为目录下的文件增加内容，不能删除和新建
chattr -a 文件，只能追加数据
chattr -a 目录，只能新建不能删除
查看权限 lsattr

sudo

visudo
实际修改的是/etc/sudoers

用户名或组名(%) 被管理主机地址=(可使用的身份) 命令(绝对路径)
yt ALL= /sbin/shutdown -r now
命令越简单权力越大

六. 文件系统管理

文件系统

逻辑分区编号从5开始

常用命令

df

统计文件系统占用情况
包括序占用的空间（如文件已经删除但程序并没有释放空间）

du

文件占用空间情况
ls命令只会显示该目录以及一级子目录所占空间
-s 统计总占用量
-a显示子文件占用量，默认只显示子目录
！！高负载命令
面向文件，只计算文件或目录占用空间，比df小

fsck

-a或-y 分区设备文件名

dumpefs

超级块信息，每块大小

挂载命令

mount 查询已经挂载设备，-l显示卷标名称
-a 依据配置文件/etc/fstab内容自动挂载
-o 特殊挂载

挂载光盘与u盘

建立挂载点 mkdir /mnt/cdrom/
挂载光盘mount -t iso9660 /dev/cdrom /mnt/cdrom
取消挂载 umont /mnt/cdrom

fdisk -l查看u盘设备文件名
建立挂载点 mkdir /mnt/usb/
挂载光盘mount -t vfat /dev/设备名 /mnt/usb
取消挂载 umont /mnt/usb

支持NTFS文件系统

安装插件 ntfs-3g

fdisk分区

/etc/fstab 文件修复

分配swap分区

free 查看内存与swap分区
